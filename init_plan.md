# Godot MCP Server

## Goal
MCP server providing accurate Godot knowledge via RAG + structured lookups. Compatible with Cursor, Claude, VS Code, etc.

## Sources
- Official Godot docs
- Demo projects
- Release notes/changelogs

## MCP Tools

### RAG
- search_docs(query)
- find_examples(query)
- related_docs(topic)

### Structured
- lookup_class(name)
- lookup_method(class, method)
- lookup_property(class, property)
- lookup_signal(class, signal)
- lookup_enum(class, enum)
- lookup_constant(class, constant)
- search_symbols(query)
- show_inheritance(class)
